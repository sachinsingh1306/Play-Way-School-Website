# 🎓 Play Way School Website

A modern and responsive **School Website** designed to provide essential information about the institution, including courses, faculty, announcements, and contact details. This project aims to create a user-friendly digital presence for schools.

---

## 🚀 Features

- 🏫 About School (Vision, Mission, Overview)
- 📚 Courses / Classes Information
- 🧑‍🏫 Faculty Details
- 📰 News & Announcements
- 🖼️ Image Gallery
- 📞 Contact Form
- 📱 Fully Responsive Design
- ⚡ Fast and Clean UI

---

### Frontend
